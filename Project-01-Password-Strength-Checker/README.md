# 🔐 Project 01 - Password Strength Checker

A secure, real-time password analysis tool developed as part of the **DecodeLabs Cyber Security Internship**. The application evaluates password strength based on cybersecurity best practices and provides users with actionable feedback to create stronger passwords.

---

## 🚀 Features

- ✅ Real-time password strength analysis
- ✅ Checks password length
- ✅ Detects uppercase and lowercase letters
- ✅ Detects numbers and special characters
- ✅ Identifies common and weak passwords
- ✅ Displays password strength (Weak, Medium, Strong)
- ✅ User-friendly interface
- ✅ Built with Python and Flask

---

## 📥 Installation

### Clone the repository

```bash
git clone https://github.com/amin-devsec/DecodeLabs-Cyber-Security-Internship.git
cd DecodeLabs-Cyber-Security-Internship/Project-01-Password-Strength-Checker
```

### Create a virtual environment (Optional)

```bash
python -m venv venv
```

### Activate the virtual environment

**Windows**

```bash
venv\Scripts\activate
```

**Linux/macOS**

```bash
source venv/bin/activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```

# 🔐 Project 02 - Basic Encryption & Decryption

A simple encryption tool developed as part of the **DecodeLabs Cyber Security Internship**. This application demonstrates the fundamentals of cryptography by implementing the **Caesar Cipher** algorithm to encrypt and decrypt text, helping users understand the basics of data confidentiality.

---

## 🚀 Features

- ✅ Encrypt plain text using the Caesar Cipher algorithm
- ✅ Decrypt encrypted text back to its original form
- ✅ User-defined shift key
- ✅ Supports uppercase and lowercase letters
- ✅ Preserves spaces, numbers, and special characters
- ✅ Simple command-line interface
- ✅ Input validation for the shift key

## 🔒 How It Works

1. Enter the message you want to encrypt.
2. Enter an integer shift key.
3. The program encrypts the message using the Caesar Cipher.
4. The encrypted text is immediately decrypted using the same key.
5. Both the encrypted and decrypted outputs are displayed.
